# MedCare – Medical Recommendation Platform

## Features
- Symptom-based chatbot
- Budget & city-wise doctor recommendation
- Doctor self-registration & login
- Medicine store with cart
- Frontend-only startup architecture

## Tech Stack
- React (Vite)
- React Router
- CSS
- localStorage

## Architecture
User → Chatbot → Specialist → City/Budget Filter → Doctor Selection → Medicine Purchase

## Note
This project is frontend-only and uses mock data for demonstration.
